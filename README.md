## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

1. Clone the repository
2. Run `pnpm install` to install the dependencies
3. Run `pnpm dev` to start the development server
4. Open your browser and navigate to `http://localhost:5173/weather-forecast-react-vite/`
